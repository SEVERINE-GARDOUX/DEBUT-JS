Agir sur les éléments

On peut sélectionner les éléments, soit par leurs balises, soit par leurs sélecteurs css, soit par leurs attributs

(id, class, name)

Pour cela on utilisera :

id = document.getElementById, class = document.getElementsByClassName

name = document.getElementsByName

Par sa balise, on utilise document.getElementsByTagName.

Manipuler les élements


On peut utliser soit innerHTML soit textContent

InnerHTML représente le contenu html d'un élément. Sa valeur contient des balises Html et son contenu est interprété par le navigateur.

textContent représente le contenu textuel d'un élément, sa valeur ne contient pas de balises et n'est pas interprété par le navigateur.

On peut manipuler les propriétés css, pour cela il faudra remplacer les propriétes avec tiret comme suit :

font-size devient fontSize, background-color devient backgroundColor

On doit faire du style en js comme suit : (il n'est pas conseiller de faire du style en js)

element.style.backgroundColor ='red'
