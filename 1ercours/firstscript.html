Les variables

Une variable est un local de stockage.

Une variable peut changer de valeur

Les Constantes

Les constantes, permettent de stocker des informations qui ne changeront pas

Les Booléens

Choix uniquement entre 2 valeurs. Soit vrai ou faux, noir ou blanc ...

Les types de données

On peut mettre typeof(nom de la variable)

Ca nous affichera le type de la variable. Soit "number", "string, ect..."

Les données des tableaux sont entre []

Les valeurs des objets sont entre { }. Les objets sont comme une fiche nominative

!! En js, on écrit le nombre de nos variables sans les mettre entre ' ' ou " ".

Si nous voulons que notre nombre soit pris en charge comme texte on le mettra entre ' ' ou " "
